# Aktivasi Windows dengan HWID

<details>
<summary><b>‚ÑπÔ∏è Tentang Metode HWID</b></summary>

[!NOTE]
Metode **HWID** (Hardware ID) mengaktifkan Windows secara digital dan permanen dengan mengikat lisensi ke hardware komputer Anda (motherboard, CPU, dll.). Ini adalah metode aktivasi yang paling disukai karena bersifat permanen dan tidak perlu diperbarui.
</details>

## Prasyarat
- Windows 10 atau Windows 11 versi 1607 atau yang lebih baru.
- Koneksi internet yang stabil.
- .NET Framework 4.6.1 atau yang lebih baru (biasanya sudah terinstall di Windows).

## Langkah 1: Unduh HWID Activator

Unduh tool activator resmi dari link berikut.

[üì¶ Download HWID Activator](/assets/downloads/tools/hwid-activator.zip){ .md-button .md-button--primary }

**SHA256 Checksum:** `a1b2c3d4e5f6...` *(Selalu periksa halaman [Verification](../../verification/) untuk checksum terbaru)*

## Langkah 2: Jalankan Aktivasi

1.  **Ekstrak** file ZIP yang diunduh.
2.  Klik kanan pada file `HWIDActivation.exe`.
3.  Pilih **Run as administrator**.
4.  Di jendela yang terbuka, klik tombol **Activate**.
5.  Tunggu proses hingga selesai. Anda akan melihat pesan "Activation Successful".
6.  **Restart** komputer Anda.

## ‚úÖ Verifikasi Keberhasilan Aktivasi

Setelah restart, verifikasi bahwa Windows Anda telah teraktivasi secara permanen:
1.  Buka **Settings** > **Update & Security** > **Activation**.
2.  Anda akan melihat status: "**Windows is activated with a digital license linked to your Microsoft account**" (atau similar).

## üîß Pemecahan Masalah

### Aktivasi Gagal
- Pastikan Anda terhubung ke internet.
- Pastikan Anda menjalankan tool sebagai **Administrator**.
- Jika gagal, coba gunakan metode [KMS](./kmspico.md) terlebih dahulu, lalu jalankan lagi tool HWID ini.
