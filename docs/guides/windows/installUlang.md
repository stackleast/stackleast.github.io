## Melewati Persyaratan Internet dan Akun Microsoft Windows 11

Rufus menawarkan opsi untuk melewati persyaratan Internet dan Akun Microsoft untuk menginstal Windows 11. Namun, jika Anda lebih suka metode yang berbeda, ikuti langkah-langkah berikut:

1. Saat Anda melihat layar "Mari sambungkan Anda ke jaringan",
2. Tekan Shift + F10 untuk membuka command prompt.
3. Ketik `start ms-cxh:localonly` dan tekan Enter.
