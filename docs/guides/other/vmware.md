# ⚡ Aktivasi VMware Workstation

VMware Workstation adalah software virtualisasi populer. Berikut adalah panduan aktivasinya.

## Langkah 1: Download dan Install VMware

Download installer VMware Workstation dari website resmi:
[https://www.vmware.com/products/workstation-pro/workstation-pro-evaluation.html](https://www.vmware.com/products/workstation-pro/workstation-pro-evaluation.html)

Jalankan installer dan ikuti proses instalasi seperti biasa.

## Langkah 2: Gunakan License Key

Setelah instalasi selesai, Anda akan diminta memasukkan license key. Gunakan salah satu key di bawah ini:

### VMware Workstation 17 Pro Keys
- MC60H-DWHD5-H80U9-6V85M-8280D
- 4A4RR-813DK-M81A9-4U35H-06K4N
- ZZ5EU-A6VD0-48HAP-6NYZX-Z7WX0

### VMware Workstation 17 Pro Keys
- ZF3R0-FHED2-M80TY-8QYGC-NPKYF
- YF390-0HF8P-M81RQ-2DXQE-M2UT6
- ZF71R-DMX85-08DQY-8YMNC-PPHV8

Masukkan key dan klik **Continue** untuk mengaktifkan produk.

## Catatan

- Key di atas adalah untuk versi Pro. Untuk VMware Player, gunakan versi Player yang gratis.
- Key bisa kadaluarsa jika terlalu banyak digunakan. Cari key terbaru di internet jika key di atas tidak bekerja.
