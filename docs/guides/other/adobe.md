## Langkah 2: Install Adobe Application

1.  Download dan install **Adobe Creative Cloud Desktop** dari website resmi Adobe.
2.  Login dengan akun Adobe (atau buat akun gratis).
3.  Install aplikasi Adobe yang diinginkan (e.g., Photoshop) melalui Creative Cloud app.

## Langkah 3: Jalankan Patcher

1.  Tutup semua aplikasi Adobe.
2.  Ekstrak file `adobe-patcher.zip`.
3.  Jalankan file `patcher.exe` sebagai **Administrator**.
4.  Pilih aplikasi Adobe yang ingin di-activate dari list.
5.  Klik **Patch** dan tunggu hingga proses selesai.

## Langkah 4: Blok Koneksi Internet (Opsional)

Tambahkan rule di Windows Firewall untuk memblokir koneksi internet untuk aplikasi Adobe yang di-patch untuk mencegah deaktivasi.
