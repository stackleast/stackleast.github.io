# </> Aktivasi JetBrains IDE

JetBrains menyediakan IDE powerful untuk programming. Berikut cara mengaktifkannya.

## Metode: Menggunakan License Server

Ini adalah metode aktivasi yang paling sederhana dan umum digunakan.

## Langkah 1: Buka JetBrains IDE

Buka IDE JetBrains yang Anda install (e.g., IntelliJ IDEA, PyCharm).

## Langkah 2: Masuk ke Menu Activation

1.  Buka menu **Help** > **Register**.
2.  Pilih opsi **License Server**.
3.  Di field license server, masukkan alamat berikut:
    ```
    http://jetbrains.license.wangyf.com
    ```
    atau
    ```
    https://jetbrains.license.wangyf.com
    ```
4.  Klik **Activate**.

## Langkah 3: Verifikasi

Restart IDE Anda. Aktivasi应该会berhasil dan Anda dapat menggunakan semua fitur premium.

## Catatan

- Pastikan komputer terhubung ke internet saat aktivasi.
- Alamat license server bisa berubah. Jika tidak bekerja, cari "jetbrains license server 2024" di internet untuk mendapatkan server terbaru.
